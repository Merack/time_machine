[Switch to English Version](README_EN.md)

计时机制, 灵感来源于B站视频[BV1naLozQEBq](https://www.bilibili.com/video/BV1naLozQEBq)

## ✅ 主要功能

- 专注计时与动画进度展示
- 播放提示音
- 自定义专注时间, 休息时间, 微休息时间, 微休息间隔
- 自定义是否启用微休息, 正方向计时, 进度条方向
- 支持中断恢复
- 仅测试了 Android 平台
